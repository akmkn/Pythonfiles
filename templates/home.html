  What is ATE?

  Automation Test Engine is an automated tool designed to perform Sanity and Functional Testing on Siebel CRM application.
  It offers simple single clicks to run Automated Test scripts. Users can use ATE to Test their required functionalities.
  It does not matter if you know automation or not, ATE helps users to run automated test scripts in a click.
  ATE serves the purpose of making sure that the Siebel CRM system is working fine for the expected functionalities.
 
  Advantages

  Simple - ATE offers the simplest single clicks to run Automated Test Scripts without any manual intervention

  User friendly – ATE helps the user in getting reports generated and notified in an email.

  A la carte - User can choose the required Functional scenario to be executed and Test set would be created accordingly.

  HP ALM – User does not need to login into HP ALM or QC to run the test scripts.

  Test Data – User does not need to enter any test data to test the requirements. The ATE tool will set the test data as per the requirement

  Cross Environment Testing - User can execute Test cases in different environments. Example: SIT and UAT.
