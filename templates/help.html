 How to use the tool to perform execution on Test Suite?
  1. Access the automation test execution tool page

  2. Various test suites would be available for user selection in checkbox format

  3. On selection of test suite, test scripts available under the specific test suite would be listed to the user

  4. Add the needed test scripts under the test grid

  5. Provide email address in the WatchList field

  6. Click on Execute button to trigger the execution

  7. Note: Please refrain from clicking the back arrow button after form submission
